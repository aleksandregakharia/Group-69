Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

CSS is designed to enable the separation of content and presentation, including layout, colors, and fonts. This separation can:

- improve content accessibility
- provide more flexibility and control in design
- reduce complexity and repetition
- enable faster page loading

Formatting and content separation allows the same page to appear differently on screen, print, or voice. CSS supports mobile devices, screen readers, and even Braille-based devices.

The name cascading comes from the way CSS chooses which rule to apply when multiple rules match the same element. This is called cascading priority.

The CSS specifications are maintained by the World Wide Web Consortium (W3C). The MIME type is text/css. W3C provides a free CSS validator for testing styles.

CSS is used not only with HTML, but also with XML, SVG, XHTML, and other markup languages. It is also used in programs like GTK.

Syntax

CSS has a simple syntax and uses English words to define styling properties, such as color, font-size, margin, etc.
